# zsh config

## Requirements

- [zsh](https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH)
- [ohmyzsh](https://github.com/ohmyzsh/ohmyzsh)
- [nvm](https://github.com/nvm-sh/nvm)

Once you have all requirments; simply clone this repo to root `~/` and then add the following to `.zshrc`

```sh
source "~/zsh_config/.profile"
```
